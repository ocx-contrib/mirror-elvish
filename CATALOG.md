# Elvish

Elvish is a cross-platform Unix-like shell featuring an expressive programming
language with first-class support for lambdas, namespacing, structured data
(maps and lists), exception handling, and a fully programmable interactive
REPL. It runs on Linux, macOS, and Windows.

## What's included

- **elvish** (`elvish.exe` on Windows) — the shell binary. Run scripts with
  `elvish script.elv`, evaluate code inline with `elvish -c 'echo (+ 1 2)'`,
  or launch the interactive REPL with plain `elvish`.
